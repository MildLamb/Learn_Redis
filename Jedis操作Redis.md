# Jedis
Jedis是官方推荐的Java连接开发工具，使用Java操作Redis中间件。

# 使用
- 导入依赖
```xml
<dependencies>
    <!-- 导入jedis的包 -->
    <dependency>
        <groupId>redis.clients</groupId>
        <artifactId>jedis</artifactId>
        <version>3.7.0</version>
    </dependency>
    <!-- fastjson -->
    <dependency>
        <groupId>com.alibaba</groupId>
        <artifactId>fastjson</artifactId>
        <version>1.2.78</version>
    </dependency>
</dependencies>
```
